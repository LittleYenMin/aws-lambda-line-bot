# AWS Lambda LINE ChatBot
LINE BOT micro service DEMO

## How it works?
本篇程式碼，將運行在 [AWS Lambda](https://aws.amazon.com/tw/lambda/features/)
透過 AWS Lambda 我們將不用處理伺服器的設定(serverless)，輕鬆建立 LINE BOT 服務
